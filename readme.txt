Venne:CMS
=========

Requirements
------------

- PHP 5.3.0+
- MySQL 5+
- Nette Framework 2.0-beta+
- Doctrine ORM 2.1.0+


Installing
----------

Make directories 'temp', 'config.neon' and 'log' writable.

Enter to http://{pathToVenne}/www/admin/
